## 💡 Tips importantes

### Sobre Git
- Nunca comitees `.env.local`
- Ya está en `.gitignore`
- Push a `main` dispara automáticamente Vercel

### Sobre Variables de entorno
- Crear `.env.local` para desarrollo
- En Vercel: Settings → Environment Variables
- NEXT_PUBLIC_* son públicas
- Cambios requieren reiniciar servidor

### Sobre formularios
- Formspree es gratis hasta 50 submissions/mes
- Actualiza el ID en `ContactForm.tsx`
- Fallback a WhatsApp si algo falla

### Sobre despliegue
- GitHub Actions compila automáticamente
- Vercel despliega si no hay errores
- Tarda ~2-3 minutos
- Puedes ver logs en Vercel Dashboard

## 🚫 Lo que NO debes hacer

❌ **Commitear secretos o tokens**
- Ya está `.gitignore` configurado

❌ **Cambiar next.config.js sin razón**
- Está optimizado para Vercel

❌ **Subir node_modules a Git**
- `.gitignore` lo ignora automáticamente

❌ **Guardar contraseñas en código**
- Usa variables de entorno

❌ **Hacer cambios grandes sin probar localmente**
- `npm run dev` primero
