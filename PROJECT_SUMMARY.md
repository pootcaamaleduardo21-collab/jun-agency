## 🚀 Cómo empezar

### 1. **Instalación local**
```bash
cd jun-agency
npm install
npm run dev
```
Visita: `http://localhost:3000`

### 2. **Personalización rápida**
- **Cambiar copy**: Edita componentes en `/components`
- **Cambiar colores**: Edita `app/globals.css`
- **Cambiar tipografía**: Edita `app/layout.tsx`
- **Agregar imágenes**: Guarda en `public/images/`

### 3. **Despliegue**
```bash
# Opción 1: Vercel (recomendado)
npm i -g vercel
vercel

# Opción 2: Netlify (vía dashboard)
# Conectar repo en netlify.com

# Ver DEPLOYMENT.md para detalles
```

### 4. **Configurar formulario**
- Ir a [formspree.io](https://formspree.io)
- Crear cuenta
- Obtener Form ID
- Actualizar en `components/ContactForm.tsx`

### 5. **Dominio**
- Compra dominio: `jun.com.mx`
- Apunta nameservers a Vercel/Netlify
- Espera 24-48 horas

## 🎨 Dirección visual: Explicación

### **¿Por qué este diseño?**

1. **No parece agencia genérica**:
   - Minimalismo contemporáneo (no corporativo)
   - Espacios en blanco generosos
   - Tipografía moderna

2. **No parece inmobiliaria**:
   - Paleta sofisticada (no tonos tierra vibrantes)
   - Copy enfocado en estrategia digital
   - Servicios específicos (no venta de propiedades)

3. **Evita clichés tropicales**:
   - Fondo dinámico (no palmeras, playa)
   - Inspiración en topografía/arquitectura
   - Referencias sutiles a territorio

4. **Premium y sofisticado**:
   - Lujo silencioso (no gold/diamantes)
   - Contraste cuidadoso
   - Animaciones sutiles (no excesivas)

## 🔍 Estrategia SEO: Explicación

### **¿Por qué este enfoque SEO?**

1. **Keywords locales**:
   - "Agencia marketing digital Riviera Maya"
   - "Marketing real estate Playa del Carmen"
   - Búsquedas intent: agencias locales, servicios específicos

2. **Schema markup**:
   - Google entiende es agencia
   - Aparece en resultados locales
   - Credibilidad aumentada

3. **Copy estratégico**:
   - Primera sección refuerza keywords
   - Lenguaje natural (no keyword stuffing)
   - Beneficios claros

4. **Performance priorizado**:
   - Core Web Vitals afectan ranking
   - Mobile first (60% búsquedas mobiles)
   - Carga rápida = mejor experiencia

## 💡 Tips de optimización

### Para mejorar conversión
1. Cambiar color del botón primario (A/B test)
2. Ajustar copy del hero
3. Posición del formulario
4. Agregar urgencia suave ("Limitado a 5 nuevos clientes por mes")

### Para mejorar SEO
1. Crear blog (15-20 posts iniciales)
2. Generar backlinks desde directorios
3. Integrar testimonios con schema
4. Crear contenido long-form (2000+ palabras)

### Para mejorar performance
1. Comprimir imágenes más
2. Usar WebP
3. Lazy load en secciones abajo
4. Minificar CSS/JS
